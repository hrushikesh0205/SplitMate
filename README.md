🚀 About The Project

SplitMate is a smart expense-sharing platform that helps users manage group expenses, track balances, and settle debts efficiently.

Whether it's a college trip, shared apartment expenses, office events, or group outings, SplitMate automatically calculates balances and simplifies settlements between members.

⚡ How To Run Locally

# Clone Repository

git clone <https://github.com/hrushikesh0205/SplitMate>
cd SplitMate


# Backend Setup

cd server
npm install


Create `.env`

PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
JWT_EXPIRE=7d
CLIENT_URL=http://localhost:5173


Run Backend

npm run dev


# Frontend Setup

cd client
npm install


Create `.env`

VITE_API_URL=http://localhost:5000/api
VITE_SOCKET_URL=http://localhost:5000


Run Frontend

npm run dev
